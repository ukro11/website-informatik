Figma

Mit Figma erstelle ich meine Vorlagen für Websiten, die ich dann nachprogrammiere. Das Programm ist kostenlos und ermöglicht hochqualitative Bilder zu exportieren als z.B. SVG, PNG oder JPG (Bild-Dateien). Ein weiterer positiver Aspekt an Figma ist, dass es mit Vektoren arbeitet und diese Technologie die Bilder sehr scharf aussehen lässt. Das Programm wurde von Adobe entwickelt und wird empfohlen für die Bearbeitung von Logos und Vorlagen wie Websites verwendet zu werden.

Visual Studio Code

Mit Visual Studio programmiere ich nicht alles, aber es ist gut genug um Websiten zu programmieren, vor allem für Projekte mit Javascript/Nodejs. Es wurde von Microsoft entwickelt und ist auch kostenlos. Außerdem bezeichnet man solche Editoren wie Visual Studio Code als IDE. Das sind Editoren, die spezialisiert sind auf das Programmieren.
