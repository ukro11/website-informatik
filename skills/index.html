Begriffe erklärt.

Frontend/Backend

Frontend ist alles, was im Vordergrund passiert (z.B. wenn der Text die Farbe ändert). Deswegen nennt man Webdesigner auch Frontend Developer. Backend ist alles, was im Hintergrund passiert (z.B. wenn man einen Button anklickt und dieser einen Link öffnen soll).

IDE

Ein IDE ist ein Editor, der für das Programmieren gemacht wurde. Dieser soll einem vieles erleichtern oder auch einfach schöner aussehen. Man könnte auch theoretisch mit einem normalen Text Editor programmieren. Intellij Idea ist ein IDE, den ich benutze, wenn ich etwas mit Java programmieren will.

Nodejs

Nodejs ist eigentlich genau das selbe wie Javascript (Syntax, ...). Der einzige Unterschied ist, dass Nodejs mehr ausgeprägt ist und eher für das Backend gemacht wurde.
